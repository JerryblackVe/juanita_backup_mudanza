# 🖥️ Proveedores de Cloud/Virtual Machine con Pruebas Gratuitas y Créditos

*Investigación realizada: Marzo 2025* - Última actualización más reciente

## ⚡ Prueba Gratuita + Créditos Posteriores

### **🎯 Google Cloud Platform (GCP)**
- **Crédito inicial:** $300 USD
- **Duración:** 90 días después de activar la cuenta
- **Servicios:** Virtual Machines (Compute Engine), AI APIs, BigQuery, Storage, Kubernetes, etc.
- **Always Free:** servicios básicos ilimitados bajo límites mensuales
- **Enlace:** https://cloud.google.com/free

### **🔴 Amazon Web Services (AWS)**  
- **Crédito inicial:** $100 USD al registrarse
- **Crédito adicional:** Hasta $100 USD más mientras exploras servicios clave
- **Duración:** 6 meses en total
- **Servicios:** EC2, RDS, VPC, Lambda, S3, y más de 100 servicios
- **Enlace:** https://aws.amazon.com/free/

### **☁️ Microsoft Azure** ⚡
- **Crédito inicial:** $200 USD
- **Duración:** 30 días
- **Alta:** Requiere verificación con tarjeta de crédito
- **Servicios:** Virtual Machines, AKS (Kubernetes), App Service, SQL Database
- **Enlace:** https://azure.microsoft.com/pricing/purchase-options/azure-account

### **🟣 Oracle Cloud Infrastructure (OCI)**
- **Crédito inicial:** $300 USD
- **Duración:** 30 días desde la activación
- **Always Free:** VPS (2 compute instances de micro), 10TB transferencia/mes
- **Servicios:** Compute, Object Storage, Database, Networking
- **Enlace:** https://www.oracle.com/cloud/free/

## 🆓 VPS Proporcionales y Cloud Alternativos

### **🔷 DigitalOcean**

- **Duración:** 60 días desde la activación
- **Tipo de cuenta:** Primera vez creando cuenta
- **Servicios:** Droplets (VPS), Kubernetes, Managed Databases, Spaces Storage

### **⚡ Linode (Ahora Akamai Cloud)**

- **Crédito:** $100 USD

- **Servicios:** VPS (general purpose, storage optimized), bare metal, GPU instances, load balancers
- **Enlace:** https://www.vultr.com/

### **🟢 Kamatera**
- **Crédito:** $100 USD
- **Duración:** 30 días
- **Servicios:** VPS altamente personalizables, Docker-ready, +100 apps con un click
- **Enlace:** https://www.kamatera.com/free-trial/

### **🟤 UpCloud**
- **Crédito:** €25 EUR
- **Duración:** Indefinido hasta usar crédito
- **Servicios:** VPS (high-performance), Block Storage, Load Balancing
- **Enlace:** https://upcloud.com/

## ⚠️ Consideraciones Importantes

- **Verificación de Identidad:** La mayoría requiere:
  - Tarjeta de crédito (para verificación, no cargo inmediato)
  - Teléfono de referencia (no virtual por lo general)
  - Dirección física

- **Políticas por Región:** Los montos y duraciones pueden variar por país
- **Primera Cuenta:** Todos son válidos solo para primer registro
- **Auto-renovación:** Recuerda cancelar tiempo o configurar límites para evitar cargos
- **Límites:** Algunos restringen servicios de GPU o instancias grande durante prueba

## 🛟 Tips de Uso Inteligente

1. **Empieza con GCP o AWS** si necesitas máquinas potentes o muchos servicios
2. **Usa Oracle para proyectos pequeños** - tiene always free VPS post-evento
3. **DigitalOcean o Linode** son ideales para deployments rápidos y web
4. **Vultr** requiere decisión rápida por sus 14 días
5. **Alibaba Cloud** es excelente si necesitas cobertura Asia-Pacífico
6. **Compara precios post-prueba** para elegir tu proveedor permanente
